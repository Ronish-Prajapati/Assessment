index.html: Contains the HTML structure of the registration form.
style.css: CSS file for styling the form elements.
main.js: JavaScript file containing scripts for form validation and step navigation.

Description:
Step-by-step Form Structure: Divided into sections ("tabs") to collect specific types of user information.
Input Fields and Validation:
Basic information such as name, email, phone number, gender, and preferred fields of interest.
Password validation for specific criteria (e.g., length, uppercase, lowercase, special characters).
Phone number validation for a 10-digit format.
Navigation Buttons: "Previous" and "Next" buttons for easy navigation between form steps.
Progress Indicator: Circles indicating the current step of the registration process.
